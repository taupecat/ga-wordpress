Introduction to WordPress

What WordPress Is:

WordPress is a Content Management System (CMS): software that is designed to make it easier to publish content to the web.

What WordPress Isn't:

  • Operating System
  • Programming Language
  • JavaScript Framework

Alternatives to WordPress

  • Drupal
  • Joomla!
  • Adobe Experience Manager
  • Sitecore
  • Sitefinity

Why WordPress?

Powers 26% of the Internet.

  • Easy to install
  • Easy to maintain
  • Easy to use
  • Flexible
  • Strong community

WordPress.com vs. WordPress.org

WordPress.com

  • Hosted platform
  • Accounts are free (pay for extra features)
  • Maintenance is done for you
  • Can't install your own plugins or themes

WordPress.org

  • Downloadable software
  • Self-hosted
