Themes, Plugins, & Customization

What is a theme?

The theme is the presentation layer. Ideally, it should handle:

  • Fonts
  • Colors
  • Layouts
  • Responsiveness

What is a theme not?

A theme shouldn't manage data, or types of data, like:

  • Custom Post Types
  • Custom Taxonomies
  • User Management
